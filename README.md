# Customer Behavior Analysis

##  Project Overview
This project analyzes customer shopping behavior data to understand purchasing patterns, customer demographics, and sales trends. The goal is to extract meaningful insights that can help businesses improve decision-making and marketing strategies.

##  Objectives
- Understand customer purchasing behavior
- Identify top-performing product categories
- Analyze spending patterns
- Explore demographic impact on sales
- Generate business insights from data

##  Tools & Technologies Used
- Python
- Pandas
- NumPy
- power bi
- Google Colab

##  Project Structure
- data/ → Contains dataset files
- notebook/ → Contains Jupyter Notebook used for analysis
- README.md → Project documentation

##  Process Followed
1. Data Loading
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insight Generation

##  Key Analysis Performed
- Customer age distribution
- Spending by product category
- Gender-based purchase analysis
- Monthly / category-wise sales trends
- Identification of high-value customers

##  Key Insights
- Certain product categories generate higher revenue.
- Specific age groups contribute more to total sales.
- Purchasing behavior varies across demographics.

##  How to Run the Project
1. Download the repository
2. Open the notebook file in Google Colab or Jupyter Notebook
3. Run all cells to reproduce the analysis

##  Conclusion
This analysis provides actionable insights into customer behavior and helps understand patterns that can improve business strategy and customer targeting.
