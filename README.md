 CUSTOMER BEHAVIOR ANALYSIS

This repository contains a *customer behavior analysis project* implemented using Google Colab. The goal of this project is to explore and analyze customer transaction data, uncover insights into shopping patterns, and visualize customer behavior trends to support data-driven decision making.

The analysis is performed using Python libraries such as *pandas, **numpy, **matplotlib, and **seaborn*, and the code is provided in the customer_shopping_behavior.ipynb notebook.
